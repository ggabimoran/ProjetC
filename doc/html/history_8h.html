Detailed Description

Records player moves during a game.

Authors
Geoffrey DELVAL, Rémi GUIJARRO ESPINOSA, Gabriel MORAN, Clément PERESSE
Version
lot_b
Date
04.2019

Represents board.h upper layer. This class records player moves and allows to cancel or replay a move.
