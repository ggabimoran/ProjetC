Detailed Description

Linked list of previously played moves.

PlastPlayedMove and PlastSavedMove may hold different values: After the player cancels a move, PlastSavedMove still points to the same value while PlastPlayedMove points to the 'previous' move. This allows to replay a canceled move.

It is intended to be used as a static variable inside the history source file.
